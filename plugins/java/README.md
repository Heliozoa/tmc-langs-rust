## Maven

### Student file policy

All files inside `./src/main` are considered student files.

#### Example

```bash
# Student files
./src/main/file
./src/main/subdirectory/file

# Not student files
./tests/test_file
./src/not_main/file
```

## Ant

### Student file policy
